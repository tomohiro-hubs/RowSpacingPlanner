# PV Row Spacing Planner

太陽光発電所の設計支援ツール。冬至を想定した「行間隔の最小値（影がかからない推奨ピッチ）」を計算するクライアントサイドアプリケーションです。

## 主な機能

1.  **行間隔の最適化計算**:
    *   **地域選択**: 47都道府県の庁所在地データ（緯度・経度）を内蔵。
    *   **カスタム地点**: 任意の緯度・経度を入力して、特定地点の計算も可能。
    *   **自動太陽位置計算**: 緯度・経度と時刻（冬至）から、太陽高度・方位を自動計算。
2.  **詳細なパラメータ設定**:
    *   **パネル設定**: 寸法、縦段数、傾斜角、下端GL高さから上端GLを自動算出（直接入力も可）。
    *   **方位角**: パネルの方位（南、南東、南西など）を考慮した計算に対応。
    *   **安全マージン**: 割増係数または固定距離の加算に対応。
3.  **視覚化と分析**:
    *   **結果グラフ**: 9:00〜15:00までの時間変化による必要行間隔の推移をグラフ表示。最大値をハイライト。
    *   **断面図**: 計算結果をSVGによる断面図で動的に表示。
    *   **既存案検証**: 設計中の数値を入力し、OK/NGを判定。
    *   **計算式表示**: 実際の数値が入った計算式と詳細パラメータを確認可能。
4.  **完全クライアントサイド動作**:
    *   サーバー通信不要。
    *   `index.html` 単体で動作可能（ビルド不要）。

## すぐに使う (`index.html`)

プロジェクトルートにある `index.html` をブラウザで開くだけで、すべての機能を利用できます。ビルド環境は不要です。

## 開発者向け情報

ソースコードは `src/` ディレクトリにあり、Vite + React プロジェクトとして構成されています。

### 技術スタック
*   React 18
*   Tailwind CSS (CDN版 / ビルド版)
*   Chart.js (グラフ描画)
*   Babel (ブラウザ内JSX変換)

### インストールと開発
本格的な開発を行う場合：

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build
```
